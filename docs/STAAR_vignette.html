Dynamic incorporation of multiple in silico functional annotations empowers rare variant association analysis of large whole-genome sequencing studies at scale

Xihao Li and Zilin Li

2020-09-14

The STAAR package accompanies the paper “Dynamic incorporation of multiple in silico functional annotations empowers rare variant association analysis of large whole-genome sequencing studies at scale” and is designed for detecting rare-variant associations in whole-genome sequencing studies.

Note: The following results are depend on R 3.5.1.

Data set-up: Generating signal regions, covariates, family-structure

The genotype data was generated by simulating 10,000 samples for a 1 megabase (Mb) region using the calibration coalescent model (COSI). The data were generated to mimic the linkage disequilibrium structure of an African-American population by using COSI. We analyzed 30,095 low frequency and rare variants (minor allele frequeny, MAF < 0.05) in the sequence.

We simulated 2,500 families and each family has 4 individuals. The kinship matrix is generated as follows:

## kinship
grid <- 1
Npercell <- 10000
ndiv <- 1
vfam <- 0.5
N <- round(grid*grid*Npercell/ndiv)

unitmat <- matrix(0.5, 4, 4)
diag(unitmat) <- 1
unitmat[1,2] <- unitmat[2,1] <- 0
ped <- data.frame(famid = rep(as.integer(1:2500), each=4), id = as.integer(1:10000L), fa = rep(0, 10000), mo = rep(0, 10000))
for(i in 1:2500) {
  ped$fa[4*i-(0:1)] <- ped$id[4*i-3]
    ped$mo[4*i-(0:1)] <- ped$id[4*i-2]
}
kins <- makekinship(ped$famid, ped$id, ped$fa, ped$mo)

We generated two covariates, one is continous, the other one is binary.

set.seed(600)
X1 <- rnorm(N)
X2 <- rbinom(N,1,0.5)
eps <- rnorm(N)

We generated 10 functional annotations.

numVar <- dim(snploc)[1]
Z1 <- rnorm(numVar)
Z2 <- rnorm(numVar)
Z3 <- rnorm(numVar)
Z4 <- rnorm(numVar)
Z5 <- rnorm(numVar)
Z6 <- rnorm(numVar)
Z7 <- rnorm(numVar)
Z8 <- rnorm(numVar)
Z9 <- rnorm(numVar)
Z10 <- rnorm(numVar)
Z <- cbind(Z1,Z2,Z3,Z4,Z5,Z6,Z7,Z8,Z9,Z10)

We calculated the Phred score of the 10 annotations. Note that the input of annotations for STAAR function was at Phred scale (annotation_phred). In practice, the Phred score is calculated based on all variants across the genome (http://favor.genohub.org).

rank <- colRanks(Z,preserveShape = TRUE)
PHRED <- -10*log10(1-rank/dim(rank)[1])

We randomly selected a 5-kilobase (kb) signal region (genotype-phenotype association regions) across the 1-Mb region.

# Length of the sequence
maxLength <- 1000000
# Length of the signal region
sigLength <- 5000
# Location of the signal region
startloc <- sample(1:(maxLength-sigLength+1),1)
endloc <- startloc + sigLength - 1
# Extract the signal region from snploc
snplist <- which(snploc$CHROM_POS >= startloc & snploc$CHROM_POS <= endloc)
snpRegion <- snploc[snplist,]
# number of variants in the signal region
numSNPs <- dim(snpRegion)[1]
# genotype of the signal region
Geno <- genotype[,snplist]

We first randomly select 5 annotations to determine the probability of variants being causal in the region through a logistic model. We then randomly selected casual variants based on these probabilities. More details could be found in Online Methods.

# Simulate causal variants from this region
b0 <- rje::logit(0.015)
b <- rep(log(5),10)

causalprob <- apply(Z[snplist,],1,function(z){
  ind <- sample(1:10,5)
  rje::expit(b0 + b[ind] %*% z[ind])
})
isCausal <- rbinom(numSNPs,1,causalprob)

Analyzing the simulated data using STAAR

Unrelated samples

We set the effect sizes as a decreasing function of MAFs, beta=-0.13*log10(MAF). The coefficients for the causal variants were all positive. The phenotype of unrelated samples was generated through a linear model.

# effects size of covariates 
alpha0 <- 0
alpha1 <- 0.5
alpha2 <- 0.5
# effect size of causal variants
c0 <- 0.13
beta <- -c0 * log10(maf[snplist][which(isCausal==1)])
# generate phenotype data
Y <- alpha0 + alpha1 * X1 + alpha2 * X2 + as.matrix(genotype)[,snplist][,which(isCausal==1)] %*% beta + eps

We applied STAAR to analyze the simulated data set. We first fit the null model using GLM.

# Fitting STAAR null model for unrelated samples using fit_null_glm
pheno <- data.frame(Y=Y,X1=X1,X2=X2)
obj_nullmodel <- fit_null_glm(Y~X1+X2,data=pheno,family="gaussian")

We then applied STAAR to test for the association.

pvalues <- STAAR(Geno,obj_nullmodel,PHRED[snplist,],rare_maf_cutoff = 0.05)

Related samples

The phenotype of related samples was generated through a linear mixed effects model.

## random effect
randomfam <- mvrnorm(N/4, rep(0, 4), vfam * unitmat)
randomfam <- as.vector(t(randomfam))
id0 <- 1:N

# generate phenotype data
Y <- alpha0 + alpha1 * X1 + alpha2 * X2 + as.matrix(genotype)[,snplist][,which(isCausal==1)] %*% beta + randomfam + eps

Analyzing data using sparse GRM

We applied STAAR to analyze the simulated data set. Note that the genetic relatedness matrix (GRM) was a sparse matrix. We first fit the null model using GLMM.

# Fitting STAAR null model for related samples using fit_null_glmmkin
pheno <- data.frame(Y=Y,X1=X1,X2=X2,id=id0)
obj_nullmodel <- fit_null_glmmkin(Y~X1+X2,data=pheno,family=gaussian(link="identity"),id="id",kins=kins)
## [1] "kins is a sparse matrix."

We then applied STAAR to test for the association.

pvalues <- STAAR(Geno,obj_nullmodel,PHRED[snplist,],rare_maf_cutoff = 0.05)

Analyzing data using dense GRM

We started from a GRM as a dense matrix.

kins <- as.matrix(kins)

We then fit the null model using GLMM. The default would automatically convert dense GRM to sparse GRM, and fit the null model using sparse GRM.

obj_nullmodel <- fit_null_glmmkin(Y~X1+X2,data=pheno,family=gaussian(link="identity"),id="id",kins=kins)
## [1] "kins is a dense matrix."

We then applied STAAR to test for the association.

pvalues <- STAAR(Geno,obj_nullmodel,PHRED[snplist,],rare_maf_cutoff = 0.05)

The STAAR-O P-value is

pvalues$results_STAAR_O
## [1] 1.78779e-08

Note that the results of using dense GRM and sparse GRM were exactly the same, as excepted.
